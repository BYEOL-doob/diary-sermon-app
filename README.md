# Diary Sermon App (Vite + React)

Basic starter for a diary app with sermon recommendation and AI analysis.

## Included

- React + Vite structure
- local storage handling via `local-storage`
- HTTP requests via `axios`
- AI analysis integration via `openai`
- Future-ready placeholder for `puppeteer-core` (server-side)

## Environment

Create `.env`:

```env
OPENAI_API_KEY=your_openai_api_key_here
VITE_OPENAI_API_KEY=your_openai_api_key_here
```

## Run

1. Update Node.js to a modern LTS (Node 20+ recommended)
2. Install dependencies:
   - `npm install`
3. Start dev server:
   - `npm run dev`
